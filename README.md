**AnarchBot**

----------Description----------

A music bot for discord written in Python3

----------Dependencies----------

The following modules needs to be installed manually:
  1. FFMPEG (python and windows)
  2. PyNaCl (python)
  3. discord (python)
  4. discord.ext (python)
  5. yt_dlp (python)
